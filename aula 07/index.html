Aula07 - Estudo de CSS

A sigla CSS é Cascading Style Sheets é uma linguangem utilizada para estilizar Página web. Utilizando esta linguagem podemos definir:

  • Cor de fundo, texto e elementos
  • Fonte e tamanho das letras
  • Posicionamento de elementos dentro da página
  • Efeitos visuais e animações

O CSS funciona em conjuto com o HTML, enquanto o HTML diz o que aparece, o CSS diz como deve aparecer.
